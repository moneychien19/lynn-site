前端開發 | Around Home

在選擇住處時，會注意哪些地方呢？
現行租屋網站幾乎都以提供大眾運輸工具及公共設施點位為主，這些資訊可能已經足夠，然而也有這種案例：入住後發現住家附近的自行車竊案頻傳、或是發現作息時間和垃圾車經過的時間不符合...。
Around Home 整合環保署及政府資料開放平台中關於治安、環保及環境指標的相關資訊，讓使用者能輕易查詢住家附近的資訊，找到更安全、更舒適的家。


  • 使用 Figma 進行使用者介面設計，並用 JSX + CSS（SCSS 預處理器）實作
  • 使用 React 前端框架開發並串接後端 API
  • 串接 Google Maps API 以進行地圖的顯示及地址和經緯度之間的轉換


GitHub repo: https://github.com/moneychien19/Around-Home-local.git
